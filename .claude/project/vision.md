This project wants to be the design taste layer that's missing from AI-assisted frontend development. You drop it into any project and suddenly Claude doesn't just write code that works — it writes code that looks and feels intentional. Specs come out with real layout thinking, accessibility baked in, and visual decisions grounded in curated references instead of generic defaults. Reviews catch the stuff linters can't: bad hierarchy, clunky interactions, inaccessible patterns. The whole thing runs as a self-contained skill system — no dependencies, no build step, just `.claude/` config that makes every frontend task go through a design-aware pipeline before a single line of code gets written.
